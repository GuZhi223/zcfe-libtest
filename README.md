# 答题测试助手

一个油猴（Tampermonkey / ScriptCat）用户脚本，支持单选 / 多选题的自动识别与作答，内置题库并支持本地存储、远程 JSON 题库加载，以及从页面"正确答案"自动学习并回写本地题库。

## 这是什么？

为了应付学校图书馆的形式主义入馆答题。脚本已经内置了已知的题库，打开答题页面即可自动作答。如果有题目不在题库中也不用慌张——随便选一个答案提交，脚本会自动记录页面显示的正确答案并写入本地题库。之后刷新答题页面重新开始答题，之前答错的题目这次就能自动答对了。

## 功能特性

- **三种页面风格适配** — 学生可自选页面风格（`#tmNr/#tm`、`.right_part`、`.kuang-nr`），每种风格的网页元素不同，脚本均已适配
- **三层题库合并** — 内置题库 + 本地存储 + 远程 URL 题库，自动合并去重
- **自动答题** — 开启后每 300ms 轮询当前题目，命中即自动点击选项
- **自动学习** — 页面显示正确答案后自动写入本地题库，越用越准
- **模糊匹配** — 基于 Jaccard 相似度的分词匹配，阈值可调（默认 0.5）
- **悬浮面板** — 左下角显示命中 / 未命中 / 疑似不一致统计，可收起
- **多选修复** — 逐个 checkbox 点击，兼容需要逐点触发的页面
- **菜单操作** — 导入 / 导出题库 JSON、设置远程 URL、启停自动答题、调整模糊阈值等

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [ScriptCat](https://scriptcat.org/)
2. 点击下方链接安装脚本（或手动新建脚本并粘贴代码）：

   **Raw 链接：**
   ```
   https://raw.githubusercontent.com/GuZhi223/zcfe-libtest/main/答题测试助手.user.js
   ```

3. 打开答题页面，脚本将自动运行

## 使用说明

### 悬浮面板

脚本加载后页面左下角会出现面板，显示：

| 区域 | 说明 |
|------|------|
| 命中 | 在题库中找到匹配并成功作答 |
| 未命中 | 题库中无匹配记录 |
| 疑似不一致 | 命中但答案格式异常或选项不存在 |

面板可通过右上角按钮收起 / 展开，收起后点击标题栏也可展开。

### 自动答题

面板中的「自动」复选框控制是否自动点击选项。关闭后仅显示匹配结果，不自动作答。

### 油猴菜单命令

点击浏览器中的脚本图标可访问以下菜单：

| 菜单 | 功能 |
|------|------|
| 导入题库（JSON） | 粘贴 JSON 导入本地题库 |
| 导出题库（本地存储） | 将本地题库复制到剪贴板 |
| 设置题库 URL（远程） | 填写远程 JSON 题库直链 |
| 启/停自动答题 | 切换自动答题开关 |
| 设置模糊阈值 | 调整模糊匹配灵敏度（0.3~0.8） |
| 调试：复制当前题规范化题干 | 复制当前题干的标准化文本，便于手动添加题库 |
| 用内置题库覆盖当前题库 | 重置本地题库为内置题库 |
| 导出内置题库模板 | 导出内置题库 JSON 作为模板 |

### 题库 JSON 格式

```json
{
  "题目文本（规范化后）": {
    "type": "single",
    "answer": "A"
  },
  "多选题文本": {
    "type": "multi",
    "answer": ["A", "B", "D"]
  }
}
```

- `type`：`"single"` 或 `"multi"`
- `answer`：单选为单个大写字母，多选为大写字母数组

## 技术细节

- 匹配优先级：精确匹配 > 包含匹配 > Jaccard 模糊匹配
- 模糊匹配使用中文关键词分词，阈值越低匹配越宽松
- 自动学习通过页面 `#correctAns` 元素获取正确答案
- 轮询间隔 300ms，通过题干 ID（`#stid`）或规范化题干去重
